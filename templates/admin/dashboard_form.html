Embed Code
How to get the embed code from Power BI:
Open your Power BI report → File → Embed report → Website or portal → Copy the <iframe> code and paste it below.

Example:
<iframe title="My Report" src="https://app.powerbi.com/view?r=..." frameborder="0" allowFullScreen="true"></iframe>
The src and title will be extracted automatically. Only powerbi.com URLs are accepted.
